UMBLE BEANS FRONT-END WEBSITE

This project was a collaborative effort as part of a course at Jönköping University, where we created a simple front-end e-commerce website for a fictional coffee bean company called Umble Beans. The focus of the project was on the fundamentals of web development, with an emphasis on UI/UX design. Through THIS LINK you will find the GitHub pages for this project.
