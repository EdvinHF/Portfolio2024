Firley Brewed

FIRLEY BREWED REST API

As part of a course at Jönköping University, I developed a REST API using Node.js (with Express.js), Handlebars.js, and SQLite3, integrating middlewares such as Sessions, Bodyparser, and Bcrypt. The objective of the project was to build a website for a fictional beer brewing company, focusing on functionality rather than aesthetics.

The website allows users to create accounts and log in, enabling them to leave ratings for different beers. Additionally, users can be assigned an admin role, which grants them the ability to create, remove, or edit product and event pages to be able to change the contents directly from the website.Through THIS LINK you will find the GitHub pages for this project.
